FEM

FEM，Front-end Modules，即就是前端模块。硬件电路中的前端模块完成射频信号的发送放大以及接受放大，甚至包含功率检测、控制和开关的这样一个作用。

Skyworks

美国半导体器件供应商，中文名我们称为思佳讯公司；

Skyworks是一家无线半导体公司，设计并生产应用于移动通信领域的射频及完整半导体系统解决方案。

该公司向全球范围内的无线手持设备和基础设施客户供应前端模块、射频子系统及系统解决方案。

FEM产品介绍

SKY85331-11: 2.4 GHz High-Power 802.11ax WLAN
Front-End Module

SKY85331 是一款应用于2.4G WiFi的射频前端器件，我们可以直接得到的信息就是High Power，即就是大功率。

原理框图

我们可以看到如下图片就是原理框图：

  • 红色路径：表示TX路径，就是WiFi芯片发射信号，经过PA（功率放大）之后，通过SWITCH（开关），最后经过天线发射出去；
  • 绿色路径：表示RX路径，就是天线接收无线信号，经过SWITCH（开关）之后，回到WiFi芯片内部；
  • 橙色路径：表示RX路径，与上面差异在于此路径经过LNA（低噪声放大器）之后再给到WiFi芯片。
特性描述
  1. Integrated high-performance 2.4 GHz PA, LNA with bypass, and T/R switch ；
    译：集成的高性能2.4 GHz PA，具有旁路的LNA，和T / R开关；请注意并不是所有的FEM必须是这样的，也可以没有如上棕色的那个路径，看配置和需求。
  2. Fully matched input and output；
    译：全匹配输入输出，即就是表示其阻抗内部有匹配，就是50欧的输入输出阻抗；但是我们仍旧需要预留匹配元器件，因为PCB阻抗不一定准。
  3. Integrated logarithmic power detector；
    译：集成的对数功率检测；这就是完成功率的控制，需要控制发送功率，一是为了更好的线性度，二是为了满足各国规定的标准。
  4. Transmit gain: 33 dB；
    译：发送功率，这就是最大值，注意单位是dB表示输入和输出的比值。
  5. Supports 802.11ax: output power；
    译：输出功率，在802.11ax规范下。

1） +20 dBm, -43 dB DEVM, MCS11
2） +22 dBm, -40 dB DEVM, MCS11
3） +24 dBm, -35 dB DEVM, MCS9

注：+20dBm表示输出功率，-35dB DEVM表示EVM（误差向量幅度），MCS11表示码率配置表。

  1. Integrated, temperature-compensated log detector
    译：集成式温度补偿对数检测器，即就是对器件的温度补偿，因为该器件温度升高时将会导致输出功率发生波动，导致本身检测不准，而补偿可以解决这个问题。
  2. High LNA gain: 15 dB
    译：高LNA增益，即就是RX输入信号经过LNA之后的信号增益，也就是放大倍数。
  3. Small (24-pin, 3 x 5 mm) conformally shielded package
    (MSL3, 260 C per JEDEC J-STD-020)
    译：封装描述，不多讲。
功率检测与输出电压

我们可以注意到，PA的最终输出功率和检测引脚的电压是一一对应的；途中曲线是3.3V和5V供电时的差异。

操作电压

因为操作电压会影响到器件的正常工作，因此电压一定要符合规范的要求。

操作电流

操作电流是电路的保证，但是请注意一般为了减小纹波，我们一般在电路中预留CRC电路，此时一定要注意R的大小，要保证VCC-R*I要大于器件最低工作电压。
