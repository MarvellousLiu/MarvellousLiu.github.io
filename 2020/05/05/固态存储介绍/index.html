﻿### 固态硬盘介绍

常见的，我们见到的硬盘是固态硬盘和机械硬盘，近来固态硬盘似乎成为了主流，其空间容量大、速度快被人们喜爱，所以大概介绍下固态硬盘。

固态硬盘

固态硬盘-Solid State Drive。固态硬盘主要就是如下几个部分构成：

  • IO口连接器：一般是Sata、有部分是PCIe接口；

  • SSD控制器：这个很多厂商都在做，但是老美和台湾省做的比较好；

  • 闪存：这个一般就是Nnad了，当然除此之外还有一些Nand的衍生产品。

  • 外壳：外壳一般就是固定PCB板的壳子，形式多样。

简单理解就是：
接口+主控+存储器

固态硬盘表现

固态硬盘（SSD）和硬盘驱动器（HDD）的物理规格相似，但它们以非常不同的方式存储数据。硬盘驱动器和固态驱动器之间的区别在于用于存储和检索数据的技术。HDD利用磁盘作为存储，而SSD利用存储器。

二者比较：

  • SSD的性能远远优于HDD。
  • SSD比HDD具有更耐用和抗故障，因为所有部件都是固定的，不像HDD包含移动部件并且易受损坏。
  • HDD消耗的功率大于SSD消耗的功率，其原因在于HDD使用能量消耗电机来实现其功能。
  • SSD的访问时间适用于较小的文件，数据传输速率可以在100 MB / s到600 MB / s之间。相反，HDD访问时间比SSD慢10倍。它提供大约140 MB / s的数据传输速率，但是，大文件不会降低其性能。
  • SSD的寻道时间和旋转延迟优于HDD，并且启动时间也较短。
  • SSD比HDD贵，并且与相同数量的HDD相比，它提供的存储容量更少。

所以，大家看自己的需求，具体选择什么，价格因素肯定要考虑的，且请注意，机械硬盘数据恢复比固态硬盘要容易得多，固态硬盘数据一旦出现问题，则很难找回。
